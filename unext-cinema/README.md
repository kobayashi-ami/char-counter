# U-NEXT シネマ独占ガイド

**U-NEXTでしか見れない映画**（独占配信・独占見放題・独占先行）を集めた、単一HTMLの非公式ガイドアプリです。

## 特徴

- **依存ゼロ / 単一ファイル** — `index.html` をブラウザで開くだけで動作します。ビルド不要。
- **検索** — タイトル・ジャンル・説明文をインクリメンタル検索。
- **ジャンル絞り込み** — すべて / アニメ / 邦画 / 洋画・海外 / 特撮。
- **作品詳細モーダル** — あらすじ・配信形態・公開年・U-NEXT公式へのリンク。
- **配信形態バッジ** — 独占見放題 / 独占先行 / 独占配信 を色分け表示。
- ダークでシネマティックなレスポンシブUI。

## 使い方

```bash
# そのまま開く
open unext-cinema/index.html            # macOS
# または任意の静的サーバで
python3 -m http.server --directory unext-cinema 8000
```

## 収録データについて

作品データはWeb検索で確認できた実在のU-NEXT独占系タイトルをもとにしています
（例：劇場版「進撃の巨人」完結編 THE LAST ATTACK、大長編 タローマン 万博大爆発、
ラストマイル、TOKYO MER 南海ミッション、はたらく細胞、HBO作品、韓国主要局の独占配信）。

**配信状況（独占・見放題・レンタルの別）は時期により変わります。** 掲載情報は各作品の
発表・報道時点のものです。視聴前に必ず[U-NEXT公式サイト](https://video.unext.jp/)で
最新の配信状況をご確認ください。本アプリはU-NEXTの公式サービスではありません。

作品の追加・編集は `index.html` 内の `MOVIES` 配列を書き換えるだけで行えます。
