# ♪ 転調fav. — コード進行お気に入り帳

好きな転調(モジュレーション)やコード進行をコレクションしておくための、単一HTMLのWebアプリです。ビルド不要で、`index.html` をブラウザで開くだけで使えます。

## 使い方

```
open modulation-favs/index.html   # macOS
```

またはブラウザに `index.html` をドラッグ&ドロップしてください。

## 登録できる形式(3種類)

| 形式 | 内容 |
|---|---|
| 🔗 曲リンク | YouTube / Spotify のURLはページ内に埋め込み再生。その他のURLはリンクとして保存 |
| 🎧 MP3ファイル | ドラッグ&ドロップで登録。ブラウザ内(IndexedDB)に保存され、その場で再生できる |
| ✍️ 手入力 | 曲名とコード進行だけをメモとして登録 |

## 機能

- **転調ポイント**を1曲に複数登録:時間(分:秒)・転調前後のキー(メジャー/マイナー24キー)・コード進行
- 時間ボタン(▶ 1:23)を押すと**転調の5秒前から再生**(MP3・YouTube対応)
- ★お気に入り、タグ、メモ
- 検索(曲名・アーティスト・タグ・メモ・コード進行)、形式フィルタ、キーフィルタ、★のみ表示
- JSONエクスポート / インポート(バックアップ用。MP3の音声データは含まれません)

## データの保存場所

- エントリ情報: `localStorage`
- MP3の音声データ: `IndexedDB`

すべて開いたブラウザの中に保存されます。サーバーには何も送信されません。
